# agents-md

This repository exists to publish the canonical `AGENTS.md` file and the small set of supporting repository documents that make it usable in practice.

## Purpose

- Keep the AGENTS file easy to find.
- Keep the governance rules explicit and versioned.
- Keep contributor, security, and conduct expectations in one public place.

## Repository contents

- `AGENTS.md` — the source-of-truth governance file.
- `README.md` — repository overview and usage notes.
- `LICENSE` — project license.
- `CONTRIBUTING.md` — how to propose changes.
- `SECURITY.md` — how to report sensitive issues.
- `CODE_OF_CONDUCT.md` — collaboration baseline.

## How to use this repository

Treat `AGENTS.md` as the primary document. The supporting files explain how to contribute to it and how the repository is maintained.

## Editing guidance

Changes to `AGENTS.md` should stay focused, deterministic, and consistent with the repository's functional-core / imperative-shell policy.
