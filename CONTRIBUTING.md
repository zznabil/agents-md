# Contributing

This repository is intentionally small. Contributions should improve the `AGENTS.md` file or the support documents around it.

## Guidelines

- Keep changes focused on one concern.
- Explain why the change belongs in the AGENTS file or support docs.
- Prefer direct edits over broad rewrites.
- Remove obsolete text instead of preserving duplicate guidance.
- Include verification for any non-trivial update.

## Pull requests

A good pull request answers:

- What changed?
- Why is it needed?
- How was it checked?
- Does it alter the meaning of `AGENTS.md` or its support docs?

## Formatting

Follow the repository's editor and line-ending conventions. Keep Markdown readable and deterministic.
